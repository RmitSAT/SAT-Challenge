RMIT Campaign Admin

One-time setup required

This admin panel uses Firebase Realtime Database to sync campaign tags across all devices instantly. Follow the steps below once, then commit the config file — and you're done.

  1. Go to console.firebase.google.com and create a new project (or open an existing one).
  2. Click "Add app" → Web (</>). Register the app, copy the firebaseConfig object shown.
  3. Build → Realtime Database → Create database. Any region, start in test mode. Then go to Rules and paste: { "rules": { "campaignTags": { ".read": true, ".write": "auth != null" }, "studentCounter": { ".read": true, ".write": true }, "personaCounters": { ".read": true, "tbs": { ".write": true }, "scd": { ".write": true }, "sset": { ".write": true } }, "studentResults": { ".read": true, "$entryId": { ".write": true, ".validate": "newData.hasChildren(['name','persona','t']) && newData.child('name').isString() && newData.child('name').val().length <= 80 && (newData.child('persona').val()==='TBS' || newData.child('persona').val()==='SCD' || newData.child('persona').val()==='SSET') && newData.child('t').isNumber() && (!newData.hasChild('img') || (newData.child('img').isString() && newData.child('img').val().length < 400000))" } }, "sharedCards": { ".read": true, "$cardId": { ".write": "!data.exists()", ".validate": "newData.hasChildren(['img','persona','t']) && newData.child('img').isString() && newData.child('img').val().length < 300000 && (newData.child('persona').val()==='A' || newData.child('persona').val()==='B' || newData.child('persona').val()==='C') && newData.child('t').isNumber()" } }, "courses": { ".read": true, ".write": "auth != null" }, "analytics": { ".read": "auth != null", "shareClicks": { "ig": { ".write": true }, "fb": { ".write": true } } }, "triplemates": { "enabled": { ".read": true, ".write": "auth != null" }, "students": { ".read": true, "$studentId": { ".write": true, "trioId": { ".write": "!data.exists()" }, "connections": { "$connId": { ".write": true, ".validate": "newData.val() === true" } } } }, "codes": { ".read": true, "$code": { ".write": "!data.exists()", ".validate": "newData.isString()" } }, "trios": { ".read": true, "$trioId": { ".write": true } } } } }
  4. Build → Authentication → Sign-in method → Email/Password → Enable.
  5. Build → Authentication → Users → Add user. Create your admin email and password.
  6. Open firebase-config.js (in this project folder) and paste your Firebase config values into the FIREBASE_CONFIG object. Commit and push to GitHub — all done!
